# Scripture Unlocked

**God's Word. Your Blueprint for Living.**

Interactive Bible study platform featuring AI-powered teaching avatars that deliver verse-by-verse KJV studies with Hebrew/Greek word analysis using Strong's Concordance.

Created by Mark Wasmuth

## Teaching Avatars

- Moses - Patient Teacher - Verse-by-verse, line-upon-line education
- Elijah - Truth on Fire - Confrontational truth, exposing religious deception
- Deborah - Judge and Prophetess - Warm authority, family application, women in Scripture

## Tech Stack

Next.js 14, React 18, TypeScript, Tailwind CSS, Capacitor 6 (iOS + Android), Anthropic Claude API, ElevenLabs TTS

## Domains

- scriptureunlocked.app (primary)
- scripturesunlocked.com (redirect)

## Getting Started

npm install
npm run dev
